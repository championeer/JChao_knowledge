现在很多海外服务，最麻烦的根本不是贵，是你想付钱都未必付得出去。

想开个 ChatGPT Plus，卡不过；想续 Claude，怕掉订阅；想买个 X Premium，还得临时去找代付、换卡、折腾半天。钱不是问题，问题是每次到付款那一步，都像开盲盒。

尤其是国内用户，只要涉及美元支付，基本都绕不开这堆破事：卡不稳、手续费高、代付有风险，用起来特别别扭。

所以我后来干脆办了一张能长期用的Bitget U 卡，把这些支付问题一次性解决掉。

## 我为何会选择这张卡

- 这张卡本质上是一张 MasterCard 虚拟卡，由 Bitget Wallet 和瑞士持牌金融机构 Fiat24 合作推出。
- 你可以用 USDT 或 USDC 充值，线上能刷，绑微信、支付宝、Apple Pay 之后，线下也能正常用。

几个我比较在意的点：

- **0 年费、0 开卡费**，没有隐藏收费，不像有些平台先收你几十刀"开卡费"
- **每月 400U 免手续费额度**，外汇费、充值费、转换费全免。ChatGPT 20 刀 + Claude 20 刀 + 零星消费，绑绑有余
- **全球 200 万+ MasterCard 商户通用**，Amazon、ChatGPT、Claude、X 都能付
- **微信、支付宝、Apple Pay、Google Pay 都能绑**，绑完之后体验和普通银行卡一样

关于人民币消费：200 元以下和普通银行卡一样，0 手续费；200 元以上支付宝/微信会收 3%，但这是支付平台对国际卡的统一规则，不是卡本身的问题。

## 哪些人群需要这张卡

如果你有下面这些需求，基本就用得上：

- **订阅海外服务**：ChatGPT、Claude、X Premium 这类都需要稳定的外币支付
- **海淘或买数字服务**：Amazon、Steam、各类海外平台都能用
- **想把加密资产直接拿来消费**：省掉先出金再转卡的折腾
- **想线下也顺手用**：绑微信、支付宝、Apple Pay 后，日常消费也方便

如果你只是偶尔付一次，可以先不用；但只要你长期有美元支付需求，手里备一张会省很多事。

## 开卡教程

整个流程六步，照着做就行。注意不要开梯子，开了可能会导致KYC失败。

开卡前准备

先把这几样东西备好：

- 一个有效的电子邮箱
- 护照原件（有效期大于 3 个月）
- 下载身份认证 App：苹果用户在 App Store 搜 **ReadID Ready**，安卓用户参考官方安卓下载教程
- Bitget Wallet 在 Arbitrum 链上有至少 **10 USDC**（验资用的，开完卡这笔钱还是你的，爱怎么用怎么用）

**第一步：下载 Bitget Wallet**

App Store 或 Google Play 下载 Bitget Wallet，创建或导入钱包。没用过的话建议新建一个助记词钱包，助记词抄好存好。

![Image](https://pbs.twimg.com/media/HE_Gje9bkAADdth?format=jpg&name=large)

**第二步：装身份认证 App**

- iOS：App Store 搜 **ReadID Ready**
- 安卓：按官方教程操作

这个 App 是后面 KYC 认证读取护照 NFC 芯片用的，必须提前装好。

![Image](https://pbs.twimg.com/media/HE_FefTaoAEomzl?format=png&name=large)

**第三步：充值验资**

打开 Bitget Wallet，确认你在 **Arbitrum 链**上有至少 **10 USDC**。

已经有了就直接跳到第四步。没有的话，从交易所提币到 Bitget Wallet 的 Arbitrum 地址，或者通过钱包内的跨链桥转入。

注意是 Arbitrum 链上的 USDC，不是 ERC-20 主网的。

![Image](https://pbs.twimg.com/media/HE_FkZTbIAAEWZX?format=jpg&name=large)

**第四步：注册 Fiat24 银行账户**

在 Bitget Wallet App 内，底部导航栏 **钱包** → **银行卡** → **去开卡**。

按提示注册 Fiat24 银行账户，填写基本信息，签名授权。整个过程大概 2-5 分钟。

![Image](https://pbs.twimg.com/media/HE_FpMXbcAAUsGG?format=jpg&name=large)

**第五步：KYC 认证**

这步最关键，也最容易踩坑。

注册完 Fiat24 账户后，需要完成身份认证。先允许定位权限，选择 China，然后选 **Passport**（护照）验证。

![Image](https://pbs.twimg.com/media/HE_FtzzbEAEm9N_?format=jpg&name=large)

接下来填地址信息和个人资料。**地址是系统自动生成的，不要改！** 然后确认条款，选择账户用途。

![Image](https://pbs.twimg.com/media/HE_FynGaQAAIR5s?format=jpg&name=large)

填写职业信息、收入等资料，勾选接受条款，提交。

![Image](https://pbs.twimg.com/media/HE_F5WYaUAAXfRs?format=jpg&name=large)

提交后进入证件验真环节，由 ReadID 完成：**护照扫描 → 检查照片页 → NFC 芯片读取 → 人脸识别**。

全部通过后会收到 Fiat24 的确认邮件，卡就开通了。

![Image](https://pbs.twimg.com/media/HE_F-iRa0AA-gHA?format=jpg&name=large)

**第六步：设置支付币种**

KYC 通过后，给卡充值，然后在银行卡界面点 **更多** → **卡片管理**，把 Direct Pay 的四个币种都打开，再把 **Default Card Currency**（默认支付币种）设成你有余额的币种。

比如充的是 USD，默认支付币种就选 USD。

Fiat24 支持四种币种充值：**USD、CHF、EUR、RMB**。建议优先充 USD，汇兑损耗最小。

![Image](https://pbs.twimg.com/media/HE_GEnqawAA6MCT?format=jpg&name=large)

六步搞定，卡就能用了。

## 绑定各平台

卡开好了，接下来绑到你日常用的支付工具上。

**绑支付宝**：支付宝 → **我的** → **银行卡** → **添加银行卡**，输入卡号、有效期、CVV 就行。卡号信息在 Bitget Wallet 里点银行卡展开就能看到。单笔 200 以下免手续费，200 以上收 3%（支付宝对国际卡的统一规则）。

**绑 Apple Pay**：iPhone 钱包 App → 右上角 **+** → **借记卡或信用卡** → **添加其他卡片**，手动输入卡号信息。如果要验证，去邮箱收验证码。

**绑微信**：和支付宝流程基本一样。微信 → **我** → **服务** → **钱包** → **银行卡** → **添加银行卡**，填卡号信息就行。

**订阅 ChatGPT Plus**：登录 ChatGPT 官网 → 升级到 Plus → 填卡号、有效期、CVV 和账单地址。注意账单地址要和你的 IP 所在国家一致，不然可能扣款失败。

**订阅 X Premium**：登录 X → 左侧 **Premium** → 选方案 → 填卡片信息和账单地址。同样，账单地址和 IP 保持一致，推荐用瑞士或美国的 IP 和地址。

## 常见问题

**充进去的钱能转出来吗？** 目前不支持，官方说在规划中。建议用多少充多少，别一次性充太多。

**有实体卡吗？** 暂时只有虚拟卡。不过绑了微信支付宝 Apple Pay 之后线上线下都能用，实体卡的需求其实不大。

**人民币消费有限额吗？** 有。单笔不超过 3,000 元，每月不超过 50,000 元，每年不超过 600,000 元。

**护照快过期了能注册吗？** 有效期必须大于 3 个月，快过期的话建议先换新护照。

**为什么验资要 10 USDC？** 确认你是真实活跃用户。开卡后这 10 USDC 还是你的，没有冻结或限制。

**订阅 AI 服务最容易踩什么坑？** 主要就三个：节点不稳、账单地址和 IP 对不上、卡里余额卡得太死。稳一点的话，用美国或新加坡节点，账单地址和 IP 国家保持一致，余额多留 2-5 美元。

**付款被拒了怎么办？** 先检查节点、账单地址和余额。如果连续几次都失败，大概率是当前 IP 被风控了，换个更干净的节点再试。

**订阅时要不要开无痕模式？** 不是必须，但开无痕窗口、少切节点和环境，整体会更稳。
